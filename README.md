# NEXORAwebsite — Futuristic 3D Web Experience

NEXORA is a next-generation creative web experience built using modern frontend technologies.
It focuses on immersive UI, smooth animations, and interactive 3D elements to create a cinematic digital experience.

🌟 Features
⚡ Modern React + Vite setup
🎨 Futuristic UI with Tailwind CSS
🌌 3D graphics using Three.js (React Three Fiber)
🎞️ Smooth animations with GSAP & Framer Motion
🧠 Interactive and immersive user experience
📱 Responsive design
🛠️ Tech Stack
React
Vite
Tailwind CSS
Three.js / React Three Fiber
GSAP
Framer Motion

🚀 Getting Started
1. Clone the repository
git clone https://github.com/ashrithvarma2222-lgtm/NEXORAwebsite.git
2. Navigate to the project
cd NEXORAwebsite
3. Install dependencies
npm install
4. Run development server
npm run dev
📦 Build for Production
npm run build
